# FurnitureDesignArchive
Furniture Design Archive built with C# ASP.NET MVC and Entity Framework

Mock data filled with Initializer.

### Furniture Piece Controller
- Name
- Image Url
- Build Intensity
- Notes
- Completed Before

### Furniture Parts Controller
- Part Name
- Part Count
- Width
- Length
- Thickness
- Part Notes
- Part Board Foot
- Total Board Foot
